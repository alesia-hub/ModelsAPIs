API will run at the back end on this URL and following routing path:

API will connect to MongoDB to Register user. To run comparisong need to add user and ahve tokens. By default all users have 4 tokens.

  • Go to /register to register in DB.
  • Go to /refill if you want to add more tokens.
  • Go to /classify if you want to run classification on the Image.
  • Go to /compare if you want to compare two texts for similarity.
